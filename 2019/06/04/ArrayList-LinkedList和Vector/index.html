ArrayList,LinkedList和Vector

发表于 2019-06-04 | 分类于 java

   在平时的开发中，除了Map这种数据结构以外，使用的最多的应该就是List了，List作为java中的有序集合，在很多场景都有使用，今天就对其中出镜率较高的三个家伙介绍下：ArrayList，LinkedList和Vector。

ArrayList

   ArrayList也被叫做动态数组，因为它底层使用数组来保存数据，同时会根据数据量进行动态扩容。

   一般在实例化ArrayList时就会预估下数据量的大小设置一个默认值，因为扩容会耗费一定的性能，而这个是完全可以避免的。

   之前我们接触过LinkedHashMap,用来保证数据的有序性，但是List本身就是有序的了，还需要这个对象干什么呢？
   其实LinkedList和ArrayList大体上是很相似的，使用也很类似，所以他们的区别就是就在于内部实现上的不同，也就是数组和链表之间的差异。
   在添加和删除上，链表更占优势
   在修改数据和查询数据上，数组可以直接通过下标操作，更占优势。

Vector

   Vector是线程安全的对象，和ArrayList的使用只是环境不同，还有区别的地方就是ArrayaList每次扩容后大小是原来的1.5倍，Vector则是原来的两倍大小。

总结

   如果在有多个线程的环境下使用list，可以使用Vector,其他环境可以选择ArrayList和LinkedList，ArrrayList在查询数据和修改数据上更占优势，LinkedList在添加数据和删除数据还是那个更占优势，同时操作方式也更多样。
